

A simple Java project to **check password strength** and **generate strong passwords**.

---



- **Check Password Strength**: Rate passwords (1–10) based on:
  - Length ≥ 8  
  - Uppercase & lowercase letters  
  - Numbers & symbols  

- **Generate Strong Passwords**: Create random passwords with at least:
  - 1 uppercase, 1 lowercase, 1 number, 1 symbol  
  - User-defined length (≥ 8)
